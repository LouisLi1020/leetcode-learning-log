# 180 天 · 六個月 · 大計畫

> 畢業前六個月目標：取得 **SWE / IT** 相關 offer。刷題、投遞、學習、修正方向並行。

---

## Month 1 — 基礎掃蕩（DSA Fundamentals）

**重點**

- 數組 / 字串、哈希表（Arrays / Strings / Hash Maps）
- Two Pointers / Sliding Window
- Linked List 基礎

**目標**

- 完成約 **40–60** 題（Easy → 部分 Medium）
- 每題記錄思路、時間與空間複雜度

**做法**

- 每天至少 **1–2** 題
- 解完後立即總結 **pattern** 與常見陷阱

---

## Month 2 — 樹 / 遞迴 / 排序 / 二分

**重點：** Binary Tree / BST · DFS / BFS · Binary Search · Recursion  

**目標：** 約 **50** 題；練習手寫 DFS / BFS

---

## Month 3 — 進階結構 / Backtracking

**重點：** Backtracking · Heap · Trie · Union-Find  

**目標：** 組合與生成題；開始穩定 Medium；**Mock interview**

---

## Month 4 — Dynamic Programming

**重點：** 背包、序列型、狀態機等標準 pattern  

**目標：** 約 **40–60** 題 DP（理解 pattern 比題量重要）

---

## Month 5 — 圖論 + 複習 + Mock

**重點：** Graph（BFS / DFS / 最短路 / Topo Sort）· 更難 Medium · 前四個月題型複習  

**活動：** 每週至少 **1** 次模擬面試；複習筆記

---

## Month 6 — Full Interview Prep

**重點：** Coding 總整理 · Behavioral（STAR）· 入門 System Design  

**活動：** 模擬面試 **2–3** 次/週；目標公司題；白板講解

---

## 每日範例（可彈性調整）

| 時間 | 任務 |
|------|------|
| 0–15 min | 複習前一天筆記 |
| 15–60 min | 解題（1–2 題） |
| 60–90 min | 寫下思路並模擬講解 |
| 90–120 min | 複習錯題 |

## Coding 面試提醒

1. 先理清題意再寫 code  
2. 用白板或紙練習講解  
3. 明確說出 **time & space complexity**

---

## Behavioral（STAR）

| 字母 | 含義 |
|------|------|
| S | Situation（情境） |
| T | Task（任務） |
| A | Action（行動） |
| R | Result（結果） |

準備 **6–8** 個故事（deadline、debug、合作衝突等）並練習自然講述。
