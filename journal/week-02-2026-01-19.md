---

### LC 238 · Product of Array Except Self

| 項目 | 內容 |
|------|------|
| Pattern | Prefix/suffix product |
| 難度 | Medium |

**題意：** 回傳陣列 `answer`，其中 `answer[i]` 為除 `nums[i]` 外其餘元素乘積；**不可用除法**。

**核心：** 左掃建 **prefix** 到 `answer`；右掃乘 **suffix**。輸出陣列不計入額外空間。

| 變數 | 意義 |
|------|------|
| `ans[i]` | 左側乘積（第一趟）× 右側乘積（第二趟） |
| `suffix` | 從右往左累積的乘積 |

```java
public int[] productExceptSelf(int[] nums) {
    int n = nums.length;
    int[] ans = new int[n];
    ans[0] = 1;
    for (int i = 1; i < n; i++)
        ans[i] = ans[i - 1] * nums[i - 1];
    int suffix = 1;
    for (int i = n - 1; i >= 0; i--) {
        ans[i] *= suffix;
        suffix *= nums[i];
    }
    return ans;
}
```

**複雜度：** 時間 O(n) · 空間 O(1)（輸出不計）

### LC 134 · Gas Station

| 項目 | 內容 |
|------|------|
| Pattern | Greedy circuit |
| 難度 | Medium |

**題意：** 環形加油站，gas[i] 加油 cost[i] 耗油，求唯一可行起點或 -1。

**核心：** 總油量 total≥0 才有解；**tank** 累積，若 tank<0 起點改 i+1 並清零。

```java
public int canCompleteCircuit(int[] gas, int[] cost) {
    int total = 0, tank = 0, start = 0;
    for (int i = 0; i < gas.length; i++) {
        int diff = gas[i] - cost[i];
        total += diff;
        tank += diff;
        if (tank < 0) { start = i + 1; tank = 0; }
    }
    return total >= 0 ? start : -1;
}
```

**複雜度：** 時間 O(n) · 空間 O(1)

### LC 135 · Candy

| 項目 | 內容 |
|------|------|
| Pattern | Greedy two-pass |
| 難度 | Hard |

**題意：** 評分比鄰居高才得糖，每人至少 1 顆，求最少糖果總數。

**核心：** 兩趟：**左→右** 遞增則+1；**右→左** 再與右鄰比取 max，累加 candies。

```java
public int candy(int[] ratings) {
    int n = ratings.length;
    int[] candies = new int[n];
    Arrays.fill(candies, 1);
    for (int i = 1; i < n; i++)
        if (ratings[i] > ratings[i - 1]) candies[i] = candies[i - 1] + 1;
    for (int i = n - 2; i >= 0; i--)
        if (ratings[i] > ratings[i + 1])
            candies[i] = Math.max(candies[i], candies[i + 1] + 1);
    int sum = 0;
    for (int c : candies) sum += c;
    return sum;
}
```

**複雜度：** 時間 O(n) · 空間 O(n)

### LC 42 · Trapping Rain Water

| 項目 | 內容 |
|------|------|
| Pattern | Two pointers / stack |
| 難度 | Hard |

**題意：** 非負高度柱狀圖，求能接的雨水總量。

**核心：** **雙指標** l/r 維護 lMax/rMax，矮邊決定蓄水並移動。

```java
public int trap(int[] height) {
    int l = 0, r = height.length - 1, lMax = 0, rMax = 0, ans = 0;
    while (l < r) {
        if (height[l] < height[r]) {
            if (height[l] >= lMax) lMax = height[l];
            else ans += lMax - height[l];
            l++;
        } else {
            if (height[r] >= rMax) rMax = height[r];
            else ans += rMax - height[r];
            r--;
        }
    }
    return ans;
}
```

**複雜度：** 時間 O(n) · 空間 O(1)

## 2026/01/23 · Day 12

### 今日任務

- [13. Roman to Integer](https://leetcode.com/problems/roman-to-integer/)

---

### LC 13 · Roman to Integer

| 項目 | 內容 |
|------|------|
| Pattern | String scan |
| 難度 | Easy |

**題意：** 羅馬數字轉整數。

**核心：** 左到右；若 **curr < next** 則減 curr 否則加 curr。

```java
public int romanToInt(String s) {
  Map<Character, Integer> m = Map.of(
      'I',1,'V',5,'X',10,'L',50,'C',100,'D',500,'M',1000);
  int ans = 0;
  for (int i = 0; i < s.length(); i++) {
    int curr = m.get(s.charAt(i));
    if (i + 1 < s.length() && curr < m.get(s.charAt(i + 1))) ans -= curr;
    else ans += curr;
  }
  return ans;
}
```

**複雜度：** 時間 O(n) · 空間 O(1)

## 2026/01/24 · Day 13

### 今日任務

- [12. Integer to Roman](https://leetcode.com/problems/integer-to-roman/)

---

### LC 12 · Integer to Roman

| 項目 | 內容 |
|------|------|
| Pattern | Greedy subtract |
| 難度 | Medium |

**題意：** 整數轉羅馬數字（1–3999）。

**核心：** 由大到小貪心：值-符號表，while num>=val 拼接並減去。

```java
public String intToRoman(int num) {
    int[] vals = {1000,900,500,400,100,90,50,40,10,9,5,4,1};
    String[] syms = {"M","CM","D","CD","C","XC","L","XL","X","IX","V","IV","I"};
    StringBuilder sb = new StringBuilder();
    for (int i = 0; i < vals.length; i++) {
        while (num >= vals[i]) { sb.append(syms[i]); num -= vals[i]; }
    }
    return sb.toString();
}
```

**複雜度：** 時間 O(1) · 空間 O(1)

### LC 58 · Length of Last Word

| 項目 | 內容 |
|------|------|
| Pattern | String |
| 難度 | Easy |

**題意：** 回傳字串最後一個單字的長度（尾端可有空白）。

**核心：** 從尾跳過空白，再往前直到空白，計算長度。

```java
public int lengthOfLastWord(String s) {
    int i = s.length() - 1;
    while (i >= 0 && s.charAt(i) == ' ') i--;
    int end = i;
    while (i >= 0 && s.charAt(i) != ' ') i--;
    return end - i;
}
```

**複雜度：** 時間 O(n) · 空間 O(1)
