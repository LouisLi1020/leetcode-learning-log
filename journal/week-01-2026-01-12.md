### LC 88 · Merge Sorted Array

| 項目 | 內容 |
|------|------|
| Pattern | Two pointers（從尾端合併） |
| 難度 | Easy |

**題意：** 將兩個已排序陣列合併進 `nums1`（in-place），結果 non-decreasing。

**核心：** 不是在「插入」，而是比較兩邊**目前最大**的元素，把較大的放到 `nums1` 最後。從後往前填就不會覆蓋還沒用到的數字。

**指標**

| 變數 | 意義 |
|------|------|
| `p1 = m - 1` | nums1 最後一個有效元素 |
| `p2 = n - 1` | nums2 最後一個有效元素 |
| `p = m + n - 1` | nums1 下一格要填的位置 |

```java
while (p2 >= 0) {
    if (p1 >= 0 && nums1[p1] > nums2[p2]) {
        nums1[p] = nums1[p1];
        p1 -= 1;
    } else {
        nums1[p] = nums2[p2];
        p2 -= 1;
    }
    p -= 1;
}
```

`while (p2 >= 0)`：nums2 每個元素都要進 nums1；nums1 剩餘未動元素本來就在正確位置。

### LC 27 · Remove Element

| 項目 | 內容 |
|------|------|
| Pattern | Read / write 雙指標（overwrite） |
| 難度 | Easy |

**題意：** 原地移除所有等於 `val` 的元素，回傳 `k`；前 `k` 個為有效值。

**核心：** 不是「刪除」，是把「不是 val」的元素往前搬，重建陣列前段。

| 指標 | 角色 |
|------|------|
| `i` | 掃描（讀） |
| `k` | 下一個可寫入位置（寫） |

```java
int k = 0;
for (int i = 0; i < nums.length; i++) {
    if (nums[i] != val) {
        nums[k] = nums[i];
        k++;
    }
}
return k;
```

### LC 26 · Remove Duplicates from Sorted Array

| 項目 | 內容 |
|------|------|
| Pattern | 已排序陣列 + overwrite（同 LC27） |
| 難度 | Easy |

**要點：** 已排序 → 重複必連續；只需保證前 `k` 正確。

- `i`：讀
- `k`：寫（下一個非重複位置）

```java
public int removeDuplicates(int[] nums) {
    if (nums.length == 0) return 0;
    int k = 1;
    for (int i = 1; i < nums.length; i++) {
        if (nums[i] != nums[k - 1]) {
            nums[k] = nums[i];
            k++;
        }
    }
    return k;
}
```

### LC 80 · Remove Duplicates from Sorted Array II

| 項目 | 內容 |

public int removeDuplicates(int[] nums) {
    if (nums.length <= 2) return nums.length;

        if (nums[i] != nums[k - 2]) {

### LC 169 · Majority Element

| 項目 | 內容 |
|------|------|
| Pattern | Boyer–Moore 投票 |
| 難度 | Easy |

**題意：** 找出出現次數 **> n/2** 的元素（保證存在且唯一）。

`candidate` + `count`：相同則 `count++`，不同則 `count--`；`count == 0` 時換 candidate。

```java
public int majorityElement(int[] nums) {
    int candidate = 0, count = 0;
    for (int num : nums) {
        if (count == 0) candidate = num;
        count += (num == candidate) ? 1 : -1;
    }
    return candidate;
}
```

### LC 189 · Rotate Array

| 項目 | 內容 |
|------|------|
| Pattern | 三次 reverse |
| 難度 | Medium |

**題意：** 將陣列向右旋轉 `k` 次，in-place。

1. `reverse(0, n-1)` 整段  
2. `reverse(0, k-1)` 前段  
3. `reverse(k, n-1)` 後段  

（`k = k % n`）

```java
public void rotate(int[] nums, int k) {
    int n = nums.length;
    k %= n;
    reverse(nums, 0, n - 1);
    reverse(nums, 0, k - 1);
    reverse(nums, k, n - 1);
}
```

### LC 121 · Best Time to Buy and Sell Stock

| 項目 | 內容 |
|------|------|
| Pattern | 一次買賣 · 維護最低買價 |
| 難度 | Easy |

`minPrice`：目前最低買點 · `maxProfit`：目前最大利潤。每天 O(1) 判斷先買後賣。

```java
public int maxProfit(int[] prices) {
    int minPrice = prices[0], maxProfit = 0;
    for (int i = 1; i < prices.length; i++) {
        if (prices[i] < minPrice) minPrice = prices[i];
        else maxProfit = Math.max(maxProfit, prices[i] - minPrice);
    }
    return maxProfit;
}
```

### LC 55 · Jump Game

| 項目 | 內容 |
|------|------|
| Pattern | Greedy（從後往前更新 goal） |
| 難度 | Medium |

`goal` 從 `n-1` 開始；若 `i + nums[i] >= goal` 則 `goal = i`。最後 `goal == 0` 即可。

```java
public boolean canJump(int[] nums) {
    int goal = nums.length - 1;
    for (int i = nums.length - 2; i >= 0; i--) {
        if (i + nums[i] >= goal) goal = i;
    }
    return goal == 0;
}
```

---

### LC 45 · Jump Game II

| 項目 | 內容 |
|------|------|
| Pattern | BFS 思維 · `near` / `far` 視窗 |
| 難度 | Medium |

在 `[near, far]` 內找最遠可達 `farthest`，一輪結束 `jumps++`，直到 `far >= n-1`。

```java
public int jump(int[] nums) {
    int near = 0, far = 0, jumps = 0;
    while (far < nums.length - 1) {
        int farthest = far;
        for (int i = near; i <= far; i++)
            farthest = Math.max(farthest, i + nums[i]);
        near = far + 1;
        far = farthest;
        jumps++;
    }
    return jumps;
}
```

### LC 274 · H-Index

| 項目 | 內容 |
|------|------|
| Pattern | 桶排序（bucket sort） |
| 難度 | Medium |

**反思：** 一開始誤以為要找「陣列最大值」；H-Index 定義是：至少有 `h` 篇論文被引用 ≥ `h` 次。

將引用數放入 `buckets[c]`（`c >= n` 放入 `buckets[n]`），從高到低累加 `count`，第一個 `count >= i` 的 `i` 即答案。

```java
public int hIndex(int[] citations) {
    int n = citations.length;
    int[] buckets = new int[n + 1];
    for (int c : citations) buckets[Math.min(c, n)]++;
    int count = 0;
    for (int i = n; i >= 0; i--) {
        count += buckets[i];
        if (count >= i) return i;
    }
    return 0;
}
```
